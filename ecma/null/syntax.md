## [null](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/null)

## Literal (null)

---

```js
let variable = 10;
console.log(variable); //=> 10

variable = null;
console.log(variable); //=> null

variable = undefined;
console.log(variable); //=> undefined
```
